  • Internet

    (Voltar)

    A internet é uma rede global de comunicação, podendo ser vista como um conglomerado mundial de equipamentos interligados via protocolos que possibilitam a troca de dados. Surgiu de uma pesquisa sobre comutação de pacotes iniciada em 1960 nos Estados Unidos da América.

    Somente em 1969 surgiu a ARPANET, pensada para transmissão de dados militares sigilosos e interligação dos departamentos de pesquisa nos Estados Unidos. Uma rede de redes, assim chamada porque permitia conectar redes distintas e geograficamente distantes alcançando seu propósito. Anos de evolução fizeram da internet um sistema de larga escala e grande complexidade envolvendo uma gama de recursos que a tornam uma necessidade vital contemporânea para a humanidade.

  • Surgimento do HTML

    (Voltar)

    É nesse contexto de evanço tecnológico com a necessidade de organizar e padronizar a forma de interação que Tim Berners-Lee (físico britânico) criou o HyperText Markup Language (HTML).Uma linguagem de marcação focada na confecção de páginas web, como esta, interpretadas pelo nabegadores da internet. Assim, o que antes era caótico se trasnformou. Todos passaram a falar a mesma língua e, como não poderia ser diferente, se entenderem melhor.

    Todo documento HTML possui marcadores (do inglês: tags), esses marcadores são os comandos de formatação da linguagem. Um elemento é formado por um nome de marcador (tag), atributos, valores e filhos (que podem ser outros elementos ou texto). Os atributos modificam os resultados padrões dos elementos e os valores caracterizam essa mudança.

  • Tags HTML

    (Voltar)

    Seguem alguns exemplos de marcadores:

    p = marcador que define um parágrafo.
    a = marcador que define uma hiperligação.
    href = atributo que define a URL da hiperligação.
    target = atributo que define a forma como a hiperligação será aberta.
    _self = valor do atributo Target que define que a hiperligação será aberta na mesma guia.
    ⁄ =define o fechamento do elemento.

    • Atibutos de Tags

      (Voltar)

      Os atributos personalizam as tags. dão a elas forma, estilo, tipo, aparência, identificadores e mais. São os atributos que personificam as tags. Cada tag tem atributos específicos. A tag tittle por exemplo pode-se aplicar cor ao texto mas o mesmo não se aplica a qualquer tag.

      No endereço https://www.w3schools.com/tags/ref_attributes.asp é possivel verificar uma tabela contendo os atribubos disponíveis e quais tags pertencem.
